Команды Git

  • git init – инициализировать локальный Git-репозиторий в текущей папке.
  • git clone [url] – клонировать (скачать) удаленный репозиторий на локальный компьютер.
  • git add [file] – добавить изменения в файле в индекс (подготовить к коммиту).
  • git commit -m "сообщение" – зафиксировать проиндексированные изменения и создать новый коммит.
  • git status – отобразить состояние рабочей директории и индекс (посмотреть измененные файлы).
  • git push – отправить локальные коммиты в удаленный репозиторий GitHub.
  • git pull – скачать актуальные изменения из удаленного репозитория и объединить их с локальными.
